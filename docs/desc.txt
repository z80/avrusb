Мы в компании погоняли твой интерфейс - и конечно возникли предложения:

1. Главное - нужен конфигурационный текстовый файл, который можно редактировать вручную, и который меняет значения полей интерфейса.
1.1 Current limit -  в поле установки должны появляться значения из конфигурационного файла. Столько - сколько их есть в данной строке файла. Скажем если в файле стоит только одно значение - сколько не тыкай треугольник выбора - всегда будет это одно значение. Максимальное число значений - 16.
1.2 Изменене масштаба шкалы для RPM (а может это уже сделано?) - из файла
1.3 Error Messages - должны браться из файла конфигурации, а из ATMega88 читается 2 байта, при этом каждый бит дает свою ошибку (всего 16 возможных ошибок). Если ошибок нет - показывается сообщение "No errors" (тоже из файла конфигурации). 
1.4 В моторе могут быть перепутаны датчики, и как следствие направление вращения может быть изменено на обратное. Конфигурационный файл должен иметь битовое поле, изменение которого изменяет значения  направления вращения на противоположное. Скажем, нормально если в ATMega  записывается/читается 1, то это соответствует CCW. Для корявого мотора это может быть 0. Поэтому для корявого мотора должна быть возможность изменить поле в файле конфигурации, чтобы программа показывала правильно.

2. Левая сторона: 
2.1 Первое поле сверху - просто измени надписи (действия правильные): Throttle type=>Voltage 0-5V/Current 0-20mA/Software control
2.2 Давай сделаем следующим полем Throttle Mode => Torque Control/Speed Control  (вместо throttle/speed)
2.3 Дальше вниз - всё как есть - software control с рамочкой вокруг, но пожалуйста после полей внутри рамочки напиши текст: "In software control mode the above parameters override the external inputs."
2.4 Внизу где-нибудь  добавь пожалуйста ещё одно поле: Speed Limit, RPM. Максимальное значение - 15000rpm.

3. Правая сторона:
3.1 справа не должно быть треугольников у полей - все поля всегда светлые и только читаются.
3.2 Поменяй пожалуйста дополнительные температуры на External Inputs 1 and 2.
3.3 По-моему там не хватало слова RPM около индикатора? Просто посмотри сам

4. Скорости передачи. Всё медленное - 1Герц. Установка и чтение скоростей и газа и направления вращения - 5 или 10 Герц.
